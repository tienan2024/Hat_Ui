# 扩展模块接口文档 (hat_extensions.py)

> 最后更新: 2026-06-02

## 概述

`hat_extensions.py` 是与 `hat_main.py` 完全解耦的扩展模块, 提供以下功能:

1. **双模式热像仪连接** — WiFi TCP / Lepton 串口 / 模拟器
2. **MCU 串口通信** — 传感器数据上报 + 报警指令下发
3. **报警控制器** — 超温检测 + 状态管理 + 抑制机制
4. **热区追踪器** — 跨帧高温区域追踪 + 叠加绘制
5. **STM32 机器人控制串口** — 前进/后退/停止指令

## §1 ThermalCameraManager — 热像仪管理器

### 1.1 创建实例

```python
from hat_extensions import ThermalCameraManager

thermal = ThermalCameraManager(config={
    "wifi_host": "192.168.3.166",
    "wifi_port": 5001,
    "lepton_baud": 921600,
    "default_device": "wifi",  # "wifi" / "lepton" / "simulator"
})
```

### 1.2 连接设备

```python
# WiFi 模式
ok = thermal.connect_wifi()

# Lepton 串口模式
ok = thermal.connect_lepton(port="COM4")

# 切换模式
thermal.set_mode("wifi")      # "wifi" / "lepton" / "simulator"
```

### 1.3 获取帧数据

```python
frame = thermal.get_frame()
# frame = {
#     'temps': np.ndarray,       # 温度矩阵 (H x W), 支持 80x60 / 160x120
#     'rgb': np.ndarray,         # 伪彩图 (H x W x 3) uint8
#     'width': int,
#     'height': int,
#     'maxTemp': float,
#     'minTemp': float,
#     'avgTemp': float,
#     'centerTemp': float,
#     'maxPos': {'y': int, 'x': int},
# }
# 无数据时返回 None
```

### 1.4 断开连接

```python
thermal.disconnect()
thermal.disconnect_all()
```

### 1.5 状态查询

```python
thermal.is_connected()      # bool
thermal.get_mode()          # "wifi" / "lepton" / "simulator"
thermal.get_state_text()    # "WiFi 已连接" / "Lepton COM4 已连接" / "模拟数据"
```

## §2 MCUSerialBridge — MCU 串口桥接

### 2.1 协议

**MCU → 上位机 (传感器上报):**
```
$DATA,<dht11_temp>,<dht11_humi>,<ultrasonic>,<smoke>,<light>\r\n
```

**上位机 → MCU (报警指令):**
```
$ALARM,<temp>\r\n    -- 超温报警
$NORMAL,<temp>\r\n   -- 恢复正常
```

**MCU 应答:**
```
$ACK,<STATE>\r\n
```

### 2.2 使用方法

```python
from hat_extensions import MCUSerialBridge

mcu = MCUSerialBridge(baud=115200)

# 连接
ok = mcu.connect(port="COM5")

# 读取最新传感器数据
data = mcu.get_sensor_data()
# data = SensorData(dht11_temp, dht11_humi, ultrasonic, smoke, light)

# 发送报警/正常指令
mcu.send_alarm(temp=85.3)
mcu.send_normal(temp=42.1)

# 断开
mcu.disconnect()
```

### 2.3 远程 TCP 连接 (串口转网络)

当 MCU 通过工控机中继服务接入时, 使用远程 TCP 模式:

```
MCU ──串口──► 工控机(relay) ──WiFi/TCP──► PC (MCUSerialBridge, mode=remote)
```

```python
from hat_extensions import MCUSerialBridge

mcu = MCUSerialBridge(baud=115200)

# 连接远程工控机
ok = mcu.open_remote(host="192.168.3.100", port=6001)

# 查询工控机上的串口列表
ports = mcu.list_remote_ports()
# ['COM1', 'COM3', 'COM5']

# 切换工控机串口 (选择连接 MCU 的串口)
success = mcu.open_remote_port("COM3")

# 之后读取传感器数据、发送指令, 与本地串口完全一致
data = mcu.get_sensor_data()
mcu.send_alarm(temp=85.3)

# 查询连接信息
info = mcu.get_connection_info()
# {'mode': 'remote', 'host': '192.168.3.100', 'port': 6001}

# 获取连接模式
mode = mcu.get_mode()  # 'local' / 'remote' / None

# 断开 (本地和远程通用)
mcu.disconnect()
```

**特性:**
- 自动心跳保活 (每 10 秒发送 `$PING`)
- 断线自动重连 (3 秒后重试)
- `_process_line()` 本地和远程共用, 数据解析逻辑一致
- 中继控制帧 (`$PORTS`, `$CMD_OK`, `$STATUS` 等) 自动解析, 不影响 MCU 数据处理

## §3 AlarmController — 报警控制器

### 3.1 功能

- 阈值比较: 最高温度 > 阈值 → 触发报警
- 状态机: NORMAL → ALARM → SUPPRESSED → NORMAL
- 自动下发: 触发/恢复时自动调用 MCU 发送指令
- 手动取消: 抑制 3 秒后恢复检测

### 3.2 使用方法

```python
from hat_extensions import AlarmController

alarm = AlarmController(
    threshold=70.0,
    mcu_bridge=mcu,      # MCUSerialBridge 实例 (可选)
)

# 每帧调用, 传入最高温度
state = alarm.check(max_temp=85.3)
# state = AlarmState(active=True, level='alarm', color='#ff3038',
#                     text='!! 报警 85.3℃', max_temp=85.3, threshold=70.0)

# 手动取消报警
alarm.cancel()

# 修改阈值 (运行时动态调整)
alarm.set_threshold(80.0)

# 查询状态
alarm.is_active()       # bool
alarm.get_state()       # AlarmState
alarm.get_threshold()   # float
```

## §4 ThermalZoneTracker — 热区追踪器

### 4.1 功能

- 每帧分析温度矩阵中的高温区域 (自动适配 80x60 / 160x120 等分辨率)
- 跨帧追踪: 同一热区分配稳定 ID
- 轨迹记录: 记录每个热区的移动轨迹
- 叠加绘制: 在伪彩图上绘制热区标记框 + 十字准星 + 温度标注

### 4.2 使用方法

```python
from hat_extensions import ThermalZoneTracker

tracker = ThermalZoneTracker(
    threshold=70.0,       # 高温阈值 (℃)
    max_regions=6,        # 最大追踪区域数
    merge_distance=7,     # 合并距离 (像素)
    track_history=30,     # 轨迹历史长度
)

# 每帧更新 (自动记录矩阵尺寸用于坐标映射)
regions = tracker.update(temps)
# regions = [HotRegion(name, x, y, temp, level, region_id), ...]

# 在伪彩图上叠加热区标记 (坐标自动按实际矩阵尺寸映射)
annotated_rgb = tracker.draw_overlay(rgb_image)
```

### 4.3 运行时参数调整

```python
# 获取当前参数
params = tracker.get_params()
# {'threshold': 70.0, 'max_regions': 6, 'merge_distance': 7, 'track_history': 30}

# 动态修改参数 (右键菜单调用)
tracker.set_threshold(80.0)
tracker.set_max_regions(10)
tracker.set_merge_distance(10)
tracker.set_track_history(50)
```

### 4.4 查询接口

```python
tracker.get_regions()       # List[HotRegion] — 当前热区列表
tracker.get_tracks()        # Dict[int, List[Tuple[int,int]]] — 轨迹数据
tracker.get_table_data()    # List[dict] — 异常表格数据
tracker.get_region_count()  # int — 当前热区数量
tracker.get_hot_info_text() # str — 高温信息文本
```

## §5 Stm32SerialController — STM32 机器人控制串口

### 5.1 指令格式

```
前进: F:1,S:0\r\n
后退: F:0,S:1\r\n
停止: F:0,S:0\r\n
```

### 5.2 使用方法

```python
from hat_extensions import Stm32SerialController

stm32 = Stm32SerialController(port="COM3", baudrate=115200)

# 连接
ok = stm32.connect(port="COM3")

# 发送指令
stm32.send_command("F:1,S:0")  # 前进

# 查询连接状态
stm32.is_connected()  # bool

# 断开
stm32.disconnect()
```

## §7 主文件集成指南

### 7.1 导入扩展模块

在 `hat_main.py` 顶部添加:

```python
try:
    from hat_extensions import (
        ThermalCameraManager,
        MCUSerialBridge,
        AlarmController,
        ThermalZoneTracker,
        Stm32SerialController as ExtStm32Controller,
    )
    EXTENSIONS_AVAILABLE = True
except ImportError:
    EXTENSIONS_AVAILABLE = False
```

### 7.2 初始化 (在 MainWindow.__init__ 中)

```python
if EXTENSIONS_AVAILABLE:
    self.thermal_camera_mgr = ThermalCameraManager(config={...})
    self.mcu_bridge = MCUSerialBridge(baudrate=115200)
    self.alarm_ctrl = AlarmController(threshold=70.0, mcu_bridge=self.mcu_bridge)
    self.zone_tracker = ThermalZoneTracker(threshold=70.0)
    self.ext_stm32 = ExtStm32Controller(port="COM3", baudrate=115200)
    self.threshold_alarm = 70.0  # 超温报警阈值 (右键温度仪表盘可修改)
```

### 7.3 串口连接 UI

主文件中已集成三组串口连接 UI (每组独立):

| UI 区域 | 控件 | 功能 |
|---------|------|------|
| 热成像相机面板 | 设备下拉 + 串口下拉 + 刷新 + 连接 | WiFi/Lepton/模拟器切换 |
| 机器人控制面板 | 串口下拉 + 刷新 + 连接 | STM32 机器人控制串口 |
| 系统控制面板 | 串口下拉 + 刷新 + 连接 | MCU 传感器串口 |

串口列表由 `_list_serial_ports()` 直接调用 pyserial 获取, 不依赖扩展模块。

### 7.4 右键菜单设置

| 右键位置 | 功能 | 同步目标 |
|----------|------|----------|
| 温度仪表盘 (三个圆环) | 设置报警阈值 (QDoubleSpinBox) | alarm_ctrl + zone_tracker |
| 热像仪画面 | 热区追踪参数 (检测阈值/最大区域/合并距离/轨迹长度) | zone_tracker + alarm_ctrl |

两处修改的阈值会双向同步, 保证画面圈出的热区和报警触发条件一致。

### 7.5 定时器回调 (在 update_thermal 中)

```python
if EXTENSIONS_AVAILABLE and hasattr(self, 'thermal_camera_mgr'):
    frame = self.thermal_camera_mgr.get_frame()
    if frame is not None:
        self.hot_regions = self.zone_tracker.update(frame.temps)
        annotated_rgb = self.zone_tracker.draw_overlay(frame.rgb)
        self.thermal_view.set_rgb_image(annotated_rgb)
        self.update_temperature_alarm(frame.max_temp, frame.min_temp, frame.avg_temp)
        self.ext_alarm_state = self.alarm_ctrl.check(frame.max_temp)
```

### 7.6 超温报警动画

`RingGauge` 控件支持报警闪烁动画:

- `set_alarm(True)` — 启动 500ms 红色/暗色交替闪烁
- `set_alarm(False)` — 停止闪烁, 恢复正常颜色
- `set_color(color)` — 设置正常状态颜色 (报警时不覆盖)

当 `max_temp > threshold_alarm` 时, 最高温仪表盘自动闪烁。

### 7.7 关闭清理 (在 closeEvent 中)

```python
if EXTENSIONS_AVAILABLE:
    if hasattr(self, 'thermal_camera_mgr'):
        self.thermal_camera_mgr.disconnect_all()
    if hasattr(self, 'mcu_bridge'):
        self.mcu_bridge.disconnect()
    if hasattr(self, 'ext_stm32'):
        self.ext_stm32.disconnect()
```

## §8 坐标映射说明

热区追踪器 `draw_overlay` 使用实际矩阵尺寸做坐标映射, 兼容不同分辨率:

| 设备 | 温度矩阵尺寸 | 映射方式 |
|------|-------------|----------|
| Lepton 串口 | 80 x 60 | `sx = r.x * w / 80` |
| WiFi 热像仪 | 160 x 120 | `sx = r.x * w / 160` |
| 模拟器 | 80 x 60 | `sx = r.x * w / 80` |

`update()` 每帧记录实际矩阵尺寸到 `_matrix_w / _matrix_h`, `draw_overlay` 使用这些值而非硬编码常量。
