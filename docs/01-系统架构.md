## 2. 模块职责

| 模块 | 文件 | 职责 |
|------|------|------|
| 主界面 | hat_main.py | UI 渲染、可见光相机、YOLO 检测、布局管理 |
| 扩展模块 | hat_extensions.py | 热像仪接入、MCU 通信、报警控制、热区追踪 |

## 3. 数据流

### 3.1 热像仪数据流

```
WiFi 热像仪 ──TCP──► ThermalCameraMgr.request_frame()
                              │
Lepton 串口 ──UART─► ThermalCameraMgr.request_frame()
                              │
                              ▼
                    { temps, rgb, maxTemp, minTemp, avgTemp }
                              │
                              ▼
                    ThermalZoneTracker.update(temps)
                              │
                    ┌─────────┼─────────┐
                    ▼         ▼         ▼
              伪彩图显示   温度仪表盘   高温区域列表
```

### 3.2 MCU 通信数据流

**本地串口模式 (直连):**

```
上位机 ─────────────────────────────────────► MCU
         $ALARM,<temp>   (超温报警)
         $NORMAL,<temp>  (恢复正常)

MCU ─────────────────────────────────────► 上位机
         $DATA,<temp>,<humi>,<ultra>,<smoke>,<light>
         $ACK,<STATE>
```

**远程 TCP 模式 (串口转网络):**

```
改造前:  MCU ──串口──► PC (MCUSerialBridge, mode=local)
改造后:  MCU ──串口──► 工控机(relay) ──WiFi/TCP──► PC (MCUSerialBridge, mode=remote)
```

工控机运行 Node.js 中继服务 (`改造参考/relay/relay-server.js`), 做协议无关的透明转发。
PC 端通过 `MCUSerialBridge.open_remote(host, port)` 连接工控机, 数据解析逻辑与本地串口完全一致。

中继控制帧:

| 方向 | 帧 | 说明 |
|------|-----|------|
| PC→工控机 | `$PING\r\n` | 心跳保活 |
| 工控机→PC | `$PONG\r\n` | 心跳回复 |
| PC→工控机 | `$CMD,LIST_PORTS\r\n` | 查询工控机串口列表 |
| 工控机→PC | `$PORTS,COM1,COM3,...\r\n` | 串口列表响应 |
| PC→工控机 | `$CMD,OPEN,COM3\r\n` | 切换工控机串口 |
| 工控机→PC | `$CMD_OK,OPEN\r\n` / `$CMD_ERR,OPEN,...\r\n` | 切换结果 |
| 工控机→PC | `$STATUS,CONNECTED\r\n` / `$STATUS,DISCONNECTED\r\n` | MCU 串口状态 |

### 3.3 报警决策流

```
热像仪温度 ──► AlarmController.check(max_temp)
                    │
            ┌───────┴───────┐
            ▼               ▼
      max > 阈值?      max <= 阈值?
            │               │
            ▼               ▼
      发送 $ALARM     发送 $NORMAL
      UI 红色闪烁     UI 恢复正常
```

## 4. 线程模型

| 线程 | 归属 | 作用 |
|------|------|------|
| Qt 主线程 | hat_main.py | UI 渲染、定时器回调 |
| CameraWorker | hat_main.py | 可见光相机读取 + YOLO 推理 |
| YoloWorker | hat_main.py | YOLO 后台推理 |
| ThermalWiFi 拉取线程 | hat_extensions.py | WiFi 热像仪帧拉取 |
| Lepton 读取线程 | hat_extensions.py | Lepton 串口数据读取 |
| MCU 读取线程 | hat_extensions.py | MCU 串口数据解析 |
